============================
MkIV Spaceplane Parts v3.0.0
============================

This is a parts pack for Kerbal Space Program v1.4.5, providing a new stockalike spaceplane fuselage system that is designed for hauling very large parts.

============
DEPENDENCIES
============

Required:
- B9PartSwitch (2.1.0)
- ModuleManager (2.8.1)
- NearFutureProps (0.2.1)
Optional
- Community Tech Tree (v2.2+)
- Raster Prop Monitor(v0.29.2) plus ASET Props (1.4)

============
INSTALLATION
============

To install, place the GameData folder inside your Kerbal Space Program folder. If asked to overwrite files, please do so. REMOVE ANY OLD VERSIONS OF THE MARK IV SYSTEM BEFORE INSTALLING.

============
KSP-MINI-AVC
============

This mod includes version checking using MiniAVC (http://forum.kerbalspaceprogram.com/threads/79745).
If you opt-in, it will use the internet to check whether there is a new version available. Data is only read from the internet and no personal information is sent.
For a more comprehensive version checking experience, please download the KSP-AVC Plugin (http://forum.kerbalspaceprogram.com/threads/79745).
